2D Transforms: CSS transforms allows you to move, rotate, scale, and skew elements. Properties of 2D Transform: 1.Translate():It moves an element from its current position according to parametes given in x y axis tranform:translate(50px,100px); (50px=x,100px=y) 2.rotate():it rotates an element clockwise or counter-clockwise according to degree transform:rotate(20deg); transform:rotate(-20deg); (-ve values rotate counter clockwise) 3.scale():It increases or decreases the size of an element tranform:scale(2,3) two times of width 3 times of height will increases transform:scale(0.5,0.5) element to be half of its original width and height 4.scaleX(): it increases or descreases the width transform:scale(2); two times of width increases 5.scaleY(): it increases or descreases the height transform:scaleY(3) 3times of height 6.skewX(): skews an element along the x-axis by given angle transform:skewX(20deg); 7.skewY(): skews an element along with y-axis by given angle transform:skewY(20deg); 8.skew(): it skews an element along X and Y axis by given angles transform:skew(20deg, 10deg) 20deg:x-axis 10:y-axis 9.matrix(): it combines all the 2D transform methods matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY()) transform:matrix(1,-0.3,0,1,0,0)
