3D Transforms: it rotates an element in 3D properties: 1.rotateX():rotates an element in x axis transform:rotateX(150deg); 2.rotateY():rotates an element in Y axix transform:rotateY(150deg); 3.rotateZ():rotates an element in Z axis transform:rotateZ(150deg); 4.rotate3d(x,y,z,a): transform: rotate3d(1, 2, -1, 192deg); x Is a describing the x-coordinate of the vector denoting the axis of rotation which can be a positive or negative number. y Is a describing the y-coordinate of the vector denoting the axis of rotation which can be a positive or negative number. z Is a describing the z-coordinate of the vector denoting the axis of rotation which can be a positive or negative number. a Is an representing the angle of the rotation. A positive angle denotes a clockwise rotation, a negative angle a counter-clockwise one.
